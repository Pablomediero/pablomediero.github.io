ICONOS

AÑADIR ICONOS

Para poder añadir los iconos primero de todo tendremos que acceder con un link a la hoja de estilos de Boostrap

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">

Para poder insertar un icono tendremos que utilizar la etiqueta <i> con el atributo class donde introduciremos el nombre del icono deseado

Ejemplo de Inserción:

<i class="glyphicon glyphicon-cloud"></i>

Para poder cambiar de icono en este caso tendremos que cambiar la ultima palabra por la que referencie al icono deseado

EDICION DE ICONOS

Para poder editar un icono tendremos que agregar el atributo Style y lo que se quiera añadir

Ejemplos:


Tamaño de letra:


<i class="glyphicon glyphicon-cloud style="font-size:36px;"></i>


Tamaño de letra y color:


<i class="glyphicon glyphicon-cloud style="font-size:48px;color:red;"></i>


Tamaño de letra, color y sombras


<i class="glyphicon glyphicon-cloud style="font-size:60px;color:lightblue; text-shadow: 2px 2px 4px #000000;"></i>


COMBINACIONES DE ESTILOS

Tambien se pueden realizar distintas combinaciones, por ejemplo podemos combinar un icono con un elemento button

Para poder realizar una combinacion tendremos que definir un elemento cualquiera e introducir distintos atributos de estilo

Para poder editar un icono a un boton primero crearemos el botó mediante las etiquetas <Button>

Si queremos decorarlo tendremos que agregar el atributo Style y lo que se quiera añadir


Tamaño de letra y color:


<button class="glyphicon glyphicon-cloud style="font-size:18px;color:red;"></button>

En Boostrap existen diferentes versiones y hay algun pequeño cambio en la inserción de los iconos

La version anterior es Boostrap 3 y vamos a ver otras versiones existentes y las diferencias que muestran
