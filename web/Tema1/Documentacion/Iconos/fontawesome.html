ICONOS FONTAWESOME

AÑADIR ICONOS

Para poder añadir los iconos primero de todo tendremos que acceder con un link a la hoja de estilos de Font Awesome

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">

Para poder insertar un icono tendremos que utilizar la etiqueta <i> con el atributo class donde introduciremos "fa fa-address-" para referirnos a iconos

Ejemplo de Inserción:

<i class="fa fa-address-book"></i>

Para poder cambiar de icono en este caso tendremos que cambiar la ultima palabra por la que referencie al icono deseado

EDICION DE ICONOS

Para poder editar un icono tendremos que agregar el atributo Style y lo que se quiera añadir

Ejemplos:


Tamaño de letra:


<i class="fa fa-address-book" style="font-size: 24px;"> traffic </i>


Tamaño de letra y color:


<i class="fa fa-address-book" style="font-size:35px;color:lightblue;"> cloud </i>


Tamaño de letra, color y sombras

<i class="fa fa-address-book" style="font-size:30px;color:lightblue;text-shadow: 2px 2px 5px #000000;"> favorite </i>


COMBINACIONES DE ESTILOS

Tambien se pueden realizar distintas combinaciones, por ejemplo podemos combinar un icono con un elemento button

Para poder realizar una combinacion tendremos que definir un elemento cualquiera e introducir distintos atributos de estilo

Ejemplo:

Para poder editar un icono a un boton primero crearemos el botó mediante las etiquetas <Button>

Si queremos decorarlo tendremos que agregar el atributo Style y lo que se quiera añadir


Tamaño de letra y color:


<button class=""fa fa-address-book" style="font-size:18px;color:red;"></button>
